HTML & CSS Quick Reference Guide

A simple guide for common HTML tags and CSS properties.

HTML Tags

1. Heading Tags

Heading tags (h1 to h6) are HTML elements used to structure content hierarchy, with h1 representing the main title and h2-h6 indicating sub-sections. Use them to improve accessibility, SEO, and readability, ensuring a logical, non-skipped order (e.g., h1 -> h2 -> h3) while avoiding using them solely for font styling. .

2. Paragraph Tag

This element is used to define a block of text, which browsers automatically separate with a small amount of white space (margins) before and after.



There are usually about 200 words in a paragraph, but this can vary widely. Most paragraphs focus on a single idea that's expressed with an introductory sentence, then followed by two or more supporting sentences about the idea. A short paragraph may not reach even 50 words while long paragraphs can be over 400 words long, but generally speaking they tend to be approximately 200 words in length.

3. Anchor Tag

An anchor tag, represented by the (a) element, is a fundamental building block of the web used to create hyperlinks. These links allow users to navigate between different web pages, move to specific sections within the same page, or trigger actions like downloading files and sending emails.

4. Image Tag

The image tag is used to embed an image into a web page. Technically, images are not directly inserted into the document; instead, they are linked to the page using this tag, which creates a holding space for them.

5. Unordered List

An unordered list in HTML is used to group a collection of related items where the specific order does not matter. These lists are typically rendered by web browsers with bullet points (dots) rather than numbers or letters.

6. Ordered List

An ordered list in HTML is used to group a set of related items in a specific, sequential order. Unlike unordered lists that use bullet points, ordered lists automatically mark each item with a sequence marker, such as a number or a letter.

7. Div Tag

A div (division) tag in HTML is a generic container used to group other HTML elements, such as text, images, or forms. It serves as a structural building block for creating layouts, dividing a document into sections, and applying CSS styles or JavaScript interactivity.

8. Span Tag

The span tag is a generic inline container used in HTML to mark up specific parts of text or segments of a document for styling or scripting.

9. Table Tag

The table tag in HTML is the primary container used to represent tabular data, which is information organized in a two-dimensional grid of rows and columns.

10. Form Tag

The form tag in HTML is a semantic container used to collect user input on a web page and send it to a server for processing. It formally defines a section of a document where users can enter data through interactive controls like text fields, checkboxes, and buttons.

CSS Properties

  • color - Changes text color
  • background-color - Changes background
  • font-size - Changes text size
  • margin - Space outside element
  • padding - Space inside element
  • border - Adds border
  • text-align - Align text
  • width - Sets width
